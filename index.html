Object Identifier using Image Analysis

This is a object detection web app made using React JS, node JS and PostgreSQL. It predicts 5 possible outcomes of the image scanned or sent. Try it out for free now.

About our project

Our project is basically a full stack web application which consists of React JS as the front-end , Node JS as the back-end and server functions and for the database we have used PostgreSQL. This database is used to store user information like name, email, password and number of images scanned by a particular user. The application is interactive and secure. For the object detection part we have used an API called Clarifai API. This API has various machine learning models under its hood and the model which we have implemented in this app is capable of detecting over 11000 image concepts which is quite a high number and hence provides highly accurate results.
